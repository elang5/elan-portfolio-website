Quiz App

View on Github

Project Description:

This is a quiz app which utilizes the Open Trivia Database API to generate a number of questions to display to the user. The user’s score is kept in the status bar, as well as the high score of the current game. A user may play as many games in their local browser window and the high score will update accordingly. Flexbox was used for the simple, yet straightforward design interface.
